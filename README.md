# 🛒 Shopping System Project

A simple C# console application that simulates a shopping cart system.
The project demonstrates the use of data structures like **Stack** and **Collections** to manage user actions and support undo functionality.

---

## 🚀 Features

* ➕ Add item to cart
* 📦 View cart items
* ❌ Remove item from cart
* 💳 Checkout
* ↩️ Undo last action
* 🚪 Exit application

---

## 🧠 Concepts Used

* Collections in C#
* Stack (for Undo functionality)
* List (for storing cart items)
* Object-Oriented Programming (OOP)

---

## 🛠️ Technologies

* C#
* .NET

---

## ▶️ How to Run

1. Clone the repository
2. Open the project in Visual Studio
3. Build the solution
4. Run the application

---

## 💡 How It Works

* Items are stored in a cart using a collection (List).
* Each action (Add / Remove) is saved in a Stack.
* When the user selects **Undo**, the last action is reversed using the Stack (LIFO concept).
